# React 7UI

A Windows 7-inspired UI component library based on 7.css.

## Installation

```bash
npm install react-7ui
# or
yarn add react-7ui
```

## Usage

```jsx
import { Button } from "react-7ui";

function App() {
  return <Button variant="primary">Click me</Button>;
}
```

## Components

- Button - Classic Windows 7 button
- Checkbox - Windows 7 style checkbox
- ComboBox - Dropdown with input capabilities
- Menu - Classic Windows menu component
- ... (and many more)
